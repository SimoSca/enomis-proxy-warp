NOTE
====

### Publish:

```bash
npm publish
```

### Add patch:

```bash
npm version patch
```

or `major` or `minor`.

this will also add a tag into my version controll.


### Install it globally:

```bash
npm install -g enomis-proxy-warp 
```

with `-g` I ensure that the binary command `enproxy` will be available everywhere.


### Update it globally

```bash
npm update -g enomis-proxy-warp 
```
